KLARA OS — Klara's Sun

Nova Scotia healthcare navigation prototype

Patient intake, care pathway routing, and navigation for Nova Scotia. Same team — repo linked to gavinshklanka / Klara-s_Sun.

Run the app and patient UI

The app lives in cgidatahackathon/. From a clone:

  1. Open the cgidatahackathon folder.
  2. Follow cgidatahackathon/README.md: create a venv, pip install -r requirements.txt, then:

uvicorn main:app --reload --port 8000

Then open http://127.0.0.1:8000 for the patient experience.
