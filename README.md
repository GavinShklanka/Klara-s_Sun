# Klara's Sun — KLARA OS

Nova Scotia healthcare navigation prototype (patient intake, routing, care pathways).

## Run the app and UI

The app and **patient UI** live in **`cgidatahackathon/`**. From a clone:

1. Open the **`cgidatahackathon`** folder.
2. Follow **`cgidatahackathon/README.md`**: create a venv, install deps, run the server.
3. Open **http://127.0.0.1:8000** for the patient experience.

The UI is served by the FastAPI backend; you need to run the server (locally or on a Python host) to see it.
